◆ builtin_cd()

int builtin_cd ( processus_t *  cmd)

Fonction d'exécution de la commande "cd".

Fonctions spécifiques aux commandes intégrées.

Paramètres
cmdPointeur vers la structure de commande à exécuter.
Renvoie
int 0 en cas de succès, -1 en cas d'erreur.

Déplace le CWD du processus vers le répertoire spécifié dans le premier argument de la commande. Si aucun argument n'est fourni, le CWD est déplacé vers le répertoire HOME de l'utilisateur. En cas d'erreur (répertoire inexistant, permission refusée, etc.), un message d'erreur est affiché sur cmd->stderr et la fonction retourne un code d'erreur.

◆ builtin_exit()

int builtin_exit ( processus_t *  cmd)

Fonction d'exécution de la commande "exit".

Paramètres
cmdPointeur vers la structure de commande à exécuter.
Renvoie
int 0 en cas de succès, -1 en cas d'erreur.

Termine le shell avec le code de sortie spécifié dans le premier argument de la commande. Si aucun argument n'est fourni, le shell se termine avec le code de sortie 0. En cas d'erreur (argument non numérique, etc.), un message d'erreur est affiché sur cmd->stderr et la fonction retourne un code d'erreur

◆ is_builtin()

int is_builtin ( const processus_t *  cmd)

Fonction de vérification si une commande est une commande "built-in".

Paramètres
cmdStructure de commande à vérifier. (Le champ path est utilisé pour vérifier le nom de la commande.)
Renvoie
int 1 si la commande est intégrée, 0 sinon.

Les commandes intégrées sont a minima: cd, exit, export, unset, pwd.

Paramètres
cmdNom de la commande à vérifier.
Renvoie
int 1 si la commande est intégrée, 0 sinon.
