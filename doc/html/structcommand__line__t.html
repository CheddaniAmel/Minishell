Description détaillée

Structure représentant une ligne de commande.

Cette structure contient la ligne de commande complète, un tableau de structures de processus, une structure de contrôle de flux, et un tableau des descripteurs de fichiers ouverts. La structure permet à la fois d'allouer l'espace mémoire nécessaire pour les processus et de gérer le flux d'exécution entre eux. Le schéma suivant illustre la relation entre les structures:
