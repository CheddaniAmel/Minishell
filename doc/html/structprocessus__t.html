Description détaillée

Structure représentant un processus.

Cette structure contient des informations sur un processus: PID, arguments, variables d'environnement, descripteurs des IO standards, le statut de sortie, un "flag" d'exécution en arrière-plan, et les temps de démarrage/arrêt.
